Introduccion a Git

Comandos Básicos

  • git init: inicializa un repositorio git
  • git status: muestra el estado de los archivos del proyecto
  • git add nombreArchivo: agrega los archivos a area preparado (staging area)
  • git commit -m "message": guarda los atchivos preparados en el repositorio
  • git log: muestra el historial de commits que se han hecho
  • git diff nombreArchivo: muestra diferencias entra archivos de proyecto
  • git cheackout: permite moverse entre el historial de versiones
  • git config global user.email "correo"
  • git config global user.name "nombre"

    • Backtracking

  • git checkout HEAD filename: descarta cambios en el área de trabajo
  • git reset HEAD filename: descarta cambios en el área de preparado

    • Branching

  • git branch:
  • git branch branch_name:
  • git checkout branch_name:
  • git merge branch_name:
  • git branch -d branch_name:
